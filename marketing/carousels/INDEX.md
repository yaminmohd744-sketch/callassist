## How to Use

**For designers:** Each carousel file contains a full brief with per-slide copy, visual direction (hex colours, layout, typography), and a caption. Follow the visual direction specs exactly — use the Pitchr brand colour tokens (`#814ac8` purple, `#df7afe` magenta, `#000000` black, `#f0ebff` near-white). Each brief also includes motion notes for Reels/TikTok adaptation.

**For schedulers:** Post once-per-day minimum. Use the recommended posting order below — it builds audience trust before introducing the product. Post carousels natively (do not link to external slide decks). Add the caption as written and put hashtags in the first comment, not the caption body.

---

## Recommended Posting Order

Post in this order for maximum early traction. The sequence moves from pure education/validation → product reveal → proof → story → comparison.

| Order | Carousel | Why post here |
|---|---|---|
| 1st | **03 — Cold Call Survival Guide** | Pure value, no product ask — builds followers and trust immediately |
| 2nd | **04 — Talk Less, Close More** | Counterintuitive insight — high save rate, strong second impression |
| 3rd | **01 — The Freeze** | Pain-trigger carousel — validates the audience, primes them for the solution |
| 4th | **05 — Follow-Up That Gets Replies** | Another pure-value piece — reinforces you as a resource before pitching |
| 5th | **06 — Discovery Framework** | Third education piece — now you have credibility |
| 6th | **07 — Pitchr Live Demo** | First product reveal — audience is now warm, not cold |
| 7th | **11 — Why Pitchr Exists** | Founder story — humanises the brand after first product exposure |
| 8th | **02 — The $10K Call** | Stakes-based pain carousel — AE persona, high stakes |
| 9th | **08 — From 0 to Quota** | First social proof piece — success story for SDRs |
| 10th | **12 — Body Language + 10 Languages** | Second demo — reveals unique features after proof is established |
| 11th | **09 — Close Rate Dashboard** | Data-heavy social proof — targets managers, team buyers |
| 12th | **10 — Pitchr vs The Old Way** | Comparison carousel — earns more impact after audience knows Pitchr |

**Cycle note:** After the first 12, restart from #1 or introduce new carousels in the same slots. The education carousels (03, 04, 05, 06) can be reposted with fresh visuals every 60–90 days — they stay evergreen.

## Platform Notes

**Instagram:** Post carousels as-is. Use captions as written. Hashtags in first comment. Aim for 4-5 posts/week.

**TikTok:** Adapt each carousel using the "TikTok text overlay / voiceover notes" section in each file. Prioritise Carousels 01, 03, 11 for first TikTok videos (highest emotional resonance for the format). Carousel 12 has a high-performing TikTok variant (face-to-camera body language demo).
